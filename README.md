# shell

## Project setup
```
chmod +x startup.sh && ./startup.sh
```

### Runs the project
```
./run.sh

```

### Resets the data
```
./reset.sh

```

### Processes pending files
```
./refresh.sh

```

# server

### Compiles and hot-reloads for development
```
npm run nodemon
```

# client

### Compiles and hot-reloads for development
```
npm run serve
```

### Compiles and hot-reloads both the server and client for development
```
npm run dev
```
